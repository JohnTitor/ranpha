# ranpha

Generate QR code of your Wi-Fi network.

```
Usage: -p PROTOCOL -s SSID -k KEY

Available options:
    -p, --protocol <PROTOCOL>  encryption protocol, WPA2, WPA, or WEP is valid (case-insensitive),
                               any other inputs are treated as no key
    -s, --ssid <SSID>          SSID of your network
    -k, --key <KEY>            key of your network
    -h, --help                 Prints help information
```
